Profile Name:
This will be the pretty screen name that is used in plain text. You may use spaces, emoji, et cetera. Maximum 64 characters.
Profile Abbreviation:
This will be the URL-friendly name that is used in addresses and filenames. For this reason, you can only use plain ASCII letters and numbers. Hint: If you use other art gallery sites, make your abbreviation the same as your username there! Maximum 32 characters.
Avatar:
Recommended resolution 256x256; will be rescaled if outside this range. Max image size 1MB.
